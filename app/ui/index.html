Unified Fraud Review

One assistant for customer verification, transaction review, and SOP-grounded fraud reasoning.

Start with the case summary, then give one unique customer identifier. Name alone is not enough for confirmation.

1. Describe The Case Explain what happened: repeated debits, new beneficiary transfer, unusual withdrawal, phishing complaint, or unauthorized UPI activity.
2. Add One Unique Identifier Use CIF ID, account number, PAN, or registered mobile number. If only a name is entered, the chatbot will ask for a unique identifier first.
3. Add Review Window Share the date-time range to inspect. Example: 2026-04-17 00:00 to 2026-04-18 23:59 or last 3 days.
